# 🔍 Username Lookup Tool

A simple web tool to find a username across different platforms.

## 🚀 Features

* Enter any username
* Generates profile links for multiple platforms
* Opens links in a new tab

## 🛠️ Built With

* HTML
* JavaScript

## 📌 Supported Platforms

* GitHub
* Instagram
* Twitter (X)
* Reddit
* many other platforms..
  
## ▶️ How to Use

1. Open the website
<img src="images/screenshot1.png" width="500">
2. Enter a username
3. Click **Search**
<img src="images/screenshot2.png" width="500">
4. Click any link to view the profile

## 💡 Example

If you enter:
apple

You will get links like:

* https://github.com/apple
* https://instagram.com/apple
